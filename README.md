Seguindo o exemplo apresentado em seu GitHub, no qual foi mantido o ServicoJubilamento para demonstrar um 
serviço inválido nos testes, neste projeto foi utilizado o ServicoCupom com a mesma finalidade. 
Dessa forma, foi possível validar no teste o comportamento da fábrica ao encontrar uma classe 
existente que não implementa a interface IServico, resultando na exceção de "Serviço inválido".
